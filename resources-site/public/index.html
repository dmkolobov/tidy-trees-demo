tidy-trees

A library for making tidy drawings of trees.

example

usage

Provides a reagent component for drawing arbitrary trees. The tree is traversed using the provided children and branch? functions(a la zippers), and the nodes are spaced so that they do not overlap. The layout algorithm takes O(height-of-tree * nodes) time.

(ns example
  (:require [tidy-trees.reagent :refer [tidy-tree]))

  ...

  [tidy-tree tree opts]

where opts contains the following entries:

:branch?
function

When applied with a node, returns whether the node is a branch.

:children?
function

When applied with a node, returns the nodes children as a seq.

:label-branch
function

When applied with a branch node, returns a label for the branch node.

:label-term
function

When applied with a terminal node, returns a label for the terminal node.

:v-gap
number(px)

The minimum amount of space between levels.

:h-gap
number(px)

The amount of space between adjacent nodes on the same level.

:edges
keyword

Determines the way tree edges are draw.
